# Rails sample api

This is a sample rails api that generated random greeting message from given 5 preset messages

## Built with

- RoR
- PostgreSQL

### Prerequisites

- Ruby
- RoR
- PostgreSQL

## Setup

- Clone the project
- Run `bundle install` and `npm install`
- Run the `rails server` to start the server
